究極の**Endless Ragnarok 初心者ガイド**へようこそ。2026年7月9日にデベロッパーのCygamesからリリースされた、この『グランブルーファンタジー リリンク（Granblue Fantasy: Relink）』の超大型拡張DLCは、PC (Steam)、PS5、PS4、そしてSwitch 2向けに、豊富なハイエンドコンテンツを導入します。復帰したベテランプレイヤーであれ、再び空の世界へと足を踏み入れるプレイヤーであれ、進行の壁にぶつかるのを避けるためには、**Endless Ragnarokの始め方**（Endless Ragnarok how to start）を知おくことが極めて重要です。この**Endless Ragnarok 初心者ガイド**では、**Endless Ragnarokの前提クエスト**、パーティの準備方法、そして最初に解放すべき機能について順を追って解説します。

新しいエリアにアクセスする前に、DLCコンテンツの**Endless Ragnarokの解放方法**（Endless Ragnarok how to unlock）を理解する必要があります。主な入り口となるのは、本編の特定の高難易度クエストのクリアです。解放されると、ローグライトモードである「混成の領域（The Conflux）」、武器の「限界突破（Weapon Transcendence）」、および強力な「召喚システム（Summon System）」にアクセスできるようになります。**Endless Ragnarokで最初に何をするべきか**（Endless Ragnarok what to do first）迷っているなら、このガイドが役立つはずです。

## 拡張コンテンツの解放：前提条件と要件

この拡張コンテンツでの旅を始めるには、新規のセーブデータからすぐに飛び込むことはできません。従うべき特定の進行ルートがあります。まず、本編のメインストーリーキャンペーンをクリアする必要があります。キャンペーンクリア後、辺境の街フォルカまたは花都シードホルムのクエストカウンターで、難易度「Proud」のクエストが解放されます。

絶対的な**Endless Ragnarokの前提クエスト**は、Proudクエストの**「プロト・バハムトとの戦い（The Tale of Bahamut's Rage）」**です。これはプロト・バハムトとの過酷な遭遇戦であり、チームのDPS、ジーン構成、そしてギミック処理の実行力が試されます。このクエストをクリアすると、本編のエピローグが解放され、ストーリーが拡張コンテンツの出来事へと直接つながります。

エピローグが完了すると、新しい拡張コンテンツへの門が開かれたという通知が届きます。新しいゾーンの**Endless Ragnarokの解放方法**がわからない場合は、**シードホルム**または**トレダメ聖域（Tredame Sanctum）**に向かってください。これらのハブエリアには、青と金の渦巻くエネルギーでマークされた、新しくアクティブになったゲートウェイがあります。これらのゲートウェイを調べることで、拡張コンテンツの核心的な進行ループとなる、完全新規のローグライトソロモード**「混成の領域（The Conflux）」**に入ることができます。

さらに、拡張コンテンツの難易度は大幅に上昇しています。前提クエストをクリアすれば技術的にはすぐに入場可能ですが、初期ステージを生き抜くためには、キャラクターのレベルが少なくとも100、武器が最大強化（レベル150）されている必要があります。

| ステップ | 目的 | 場所 | 主な報酬 / 解放要素 |
| :--- | :--- | :--- | :--- |
| 1 | 本編キャンペーンのクリア | フォルカ / シードホルム | 難易度 Hard/Extreme クエストへのアクセス |
| 2 | 難易度 Proud への到達 | クエストカウンター | Proud クエストへのアクセス |
| 3 | 「プロト・バハムトとの戦い」のクリア | クエストカウンター (Proud) | エピローグの解放 |
| 4 | エピローグの完了 | ハブエリア | 拡張コンテンツストーリーへのアクセス |
| 5 | 混成の領域のゲートウェイを訪れる | シードホルム / トレダメ聖域 | 「混成の領域」とDLCハブの解放 |

## 最初にやること：初期進行チェックリスト

アクセス権を得た後は、膨大な新しいシステムに圧倒されるかもしれません。**Endless Ragnarokで最初に何をするべきか**を知りたい場合、主な目標は新しいキャラクターの獲得と、基本的な周回環境を整えることです。**Endless Ragnarokの始め方**を理解するには、拡張コンテンツの進行ループを把握する必要があります。

まず、新しいDLCキャラクターを獲得しましょう。この拡張コンテンツでは6人の新しいプレイアブルキャラクターが追加され、総勢28人になります。追加されるキャラクターは、**ベアトリクス**、**ユーステス**、**フラウ**、**フェディエル**、**ガランツァ**、**マギリィ**です。彼らは、新しい拡張クエストで入手できる「キャラクター解放チケット」を使用するか、キャラクターパックを所有している場合は直接加入させることができます。各キャラクターは独自のメカニクスを持っています。ベアトリクスは、攻撃速度とダメージを操作するハイリスク・ハイリターンな「デルタクロック」メカニクスを利用し、マギリィは浮遊する武器、シールド、回復による優れたサポートを提供します。

次に、**「混成の領域（The Conflux）」**に飛び込みましょう。このモードは、1回あたり約20分かかるソロ専用のローグライト体験です。これは、武器の限界突破に必要な素材を含め、ゲーム内で最も効率よく強化素材を集められるコンテンツです。この**Endless Ragnarok 初心者ガイド**に従うなら、まずは「混成の領域」を少なくとも3回クリアして、基本的なアップグレードトークンを集めることを優先してください。

3つ目に、キャラクターの**「マスタートレイト（Master Traits）」**の強化に焦点を当てましょう。キャラクターがレベル100に達すると、**「インサイト（Insight）」**、**「エッセンス（Essence）」**、**「クリュクス（Crux）」**という3つの異なるルートを解放できます。各ルートには、3つのユニークなランクパークと4つのランクスタイルアップグレードがあります。これらを解放するには、MSP（マスターポイント）と、「混成の領域」の周回でドロップする特定の素材が必要です。

| キャラクター | 武器タイプ | コアメカニクス | ロール |
| :--- | :--- | :--- | :--- |
| ベアトリクス | 大剣 | デルタクロック（攻撃速度＆低HPバフ） | 純DPS |
| ユーステス | ライフル / 手榴弾 | フレイム/サンダーチャージショット | 遠距離DPS / クラウドコントロール |
| フラウ | 格闘 | 契約者（ザ・デビルの召喚アシスト） | 近接ブルーザー / 燃焼DoT |
| フェディエル | 闇魔法 | 六竜の結界＆衰弱エリア | サポート / 持続ダメージ |
| ガランツァ | 槍 | 攻撃力スケーリング＆怯み無効攻撃 | 近接タンク / DPS |
| マギリィ | 浮遊剣 | アイアンメイデンシールド＆念動刃 | サポート / シールド付与 |

スタートを最適化するために、これらの高難易度チャレンジに挑む前に装備が整っているか確認できるよう、[Proudクエスト進行ガイド](/ja/guides/tale-of-bahamuts-rage-guide/)をチェックすることをお勧めします。

## 混成の領域を極める：ローグライトソロモード

エンドゲームの周回に向けて**Endless Ragnarokの始め方**を模索しているプレイヤーにとって、**「混成の領域（The Conflux）」**はプレイ時間の大部分を費やすことになる場所です。このモードには、シードホルムまたはトレダメ聖域にあるゲートウェイからアクセスします。

「混成の領域」では、1人のキャラクターで単独で進入します。このモードは、ランダムに生成される一連の部屋で構成されています。各エリアに入る前に、3つのランダムなパワーアップ（攻撃力上昇、クールダウン短縮、ライフスティールなど）が提示されます。その中から1つを選択し、それはエリアを退出するか周回を完了するまで有効になります。

部屋は「戦闘部屋」、「宝箱部屋」、「レアモンスター部屋」に分類されます。戦闘部屋では、制限時間内に敵のウェーブを倒す必要があります。宝箱部屋には、貴重なジーンや強化素材が入った宝箱があります。レアモンスター部屋では、大量のマスターポイントやレアな強化アイテムをドロップする「ゴールデンスライム」などの手強い敵と戦うことになります。

深層へ進むにつれて、難易度は指数関数的に上昇します。しかし、それに伴って報酬の質も向上します。周回に成功すると約20分かかり、武器の限界突破素材、マスタートレイトトークン、およびSティアのジーンが手に入ります。

深層のクリアに苦戦する場合は、維持能力の高いキャラクターの使用を検討してください。**ジークフリート**や**ヴェイン**は、防御バフやライフスティール能力を持っているため、優れた選択肢となります。あるいは、SSティアキャラクターの**サンダルフォン**は、壊滅的な光の剣波を放つ「天司長モード（Supreme Primarch Mode）」を使用して、簡単に部屋をクリアすることができます。

| 部屋タイプ | 目的 | 危険度 | 主な報酬 |
| :--- | :--- | :--- | :--- |
| 戦闘部屋 | 制限時間内にすべての敵を撃破する | 中 | 武器素材、経験値 |
| 宝箱部屋 | 宝箱を開け、環境トラップを回避する | 低 | ゴールド、ジーン強化素材 |
| レアモンスター | レアエネミー（虹スライムなど）を撃破する | 高 | マスターポイント (MSP)、混成トークン |
| ゲートウェイボス | 主要ボス（ラグナリアなど）を撃破する | 極めて高 | 限界突破素材、Sティアジーン |

このモードに最適化されたビルドの詳細な内訳については、総合的な[混成の領域周回ガイド](/ja/conflux/conflux-farming-guide/)をご覧ください。

## 武器の限界突破と召喚システム

「混成の領域」や新しい**「フェイトブレイカー（Fatebreaker）クエスト」**から素材が安定して手に入るようになったら、**「武器の限界突破（Weapon Transcendence）」**による武器の強化を開始できます。このシステムにより、最大強化された武器（レベル150）を標準の限界を超えてさらに強化できます。限界突破素材を使って**Endless Ragnarokで最初に何をするべきか**を考える際は、メインDPSキャラクターの「究極武器（Terminus）」または「クリティカル武器（Stinger）」を最優先してください。

限界突破の対象となる武器タイプは、ディフェンダー、アセンション、エグゼキューショナー、スティンガー、スタナー、ターミナスの6種類です。武器を限界突破すると、入れ替え可能なトレイトが解放されます。スロットIIは最大レベル25まで、スロットIVは最大レベル10まで強化できます。さらに、武器の限界突破を達成すると、限定のスロットEX報酬や、武器が光るユニークなエフェクトが解放されます。

拡張コンテンツで導入されたもう1つの主要なメカニクスは、**「召喚システム（Summon System）」**です。難易度「Chaos」以上のクエストでは、ルリアが戦闘を支援するために強力な星晶獣を召喚できます。各プレイヤーは編成に最大4つの召喚をセットできますが、フィールド上で同時にアクティブにできるのは1つだけです。

**ルシファー（Lucilius）**と**ローラン（Rolan）**の2つの召喚は完全に操作可能です。召喚されると、一時的に操作キャラクターと入れ替わり、壊滅的なユニーク攻撃を繰り出すことができます。さらに、召喚がアクティブな状態でパーティメンバー4人全員がリンクタイム中に奥義（SBA）を発動すると、**「プライマルバースト（Primal Burst）」**がトリガーされ、数百万ポイントのダメージを与え、ボスに長時間のデバフを付与します。

アクティブに召喚していないときでも、装備している星晶獣はキャラクターにクリティカルダメージ増加や自動復活などのパッシブステータスボーナスを提供します。

| 武器タイプ | 主な限界突破の恩恵 | 推奨キャラクター | 優先度 |
| :--- | :--- | :--- | :--- |
| ターミナス | 圧倒的な基礎攻撃力＆ダメージ上限アップ | サンダルフォン、ナルメア、ゼタ | 高 |
| アセンション | HP回復＆攻撃力アップ | グラン/ジータ（団長）、ヴェイン | 中 |
| スティンガー | クリティカル確率＆クリティカルダメージ上限アップ | ベアトリクス、ユーステス | 高 |
| ディフェンダー | 最大HP上昇＆被ダメージカット | ジークフリート、ガランツァ | 低 |

## 難易度「Chaos」の生存とジーンの最適化

**Endless Ragnarok 初心者ガイド**の旅の最後のステップは、**難易度「Chaos」**への準備です。これはゲーム内最高峰の難易度であり、**ベルゼバブ**、**ザ・ワールド**、**ラグナリア**などのボスは、適切な防御設定を欠いたキャラクターを簡単に一撃で戦闘不能（KO）にしてきます。このガイドではコア機能の**Endless Ragnarokの解放方法**を説明していますが、それらを生き抜くには装備構成の全面的な見直しが必要です。

生き残るためには、ジーン構成を最適化する必要があります。この拡張コンテンツでは、難易度「Chaos」のクエストに挑むために不可欠な、新しいSティアおよびAティアのジーンが導入されています。

*   **Sティアジーン**: **「専用ジーン+（Warpath+）」**と**「有利属性変換+（War Elemental+）」**を探しましょう。専用ジーン+はキャラクター固有のもので、ユニークメカニクスを大幅に強化します（例えば、サンダルフォンの専用ジーン+は天司長モードの持続時間を延長します）。有利属性変換+は、敵の属性に関係なくすべての攻撃が弱点属性ダメージになることを保証しつつ、サブステータスとして防御ステータスを提供します。
*   **Aティアジーン**: **「クリティカル確率 V+」**や**「アルファ/ベータ/ガンマ」**ジーンに焦点を当てましょう。これらのジーンは、奥義ダメージやリンクゲージの上昇量を高めつつ、ダメージ上限に到達することを可能にします。

ビルドを設定する際は、防御オプションを軽視しないでください。ボスのギミックによる即死を防ぐために、難易度「Chaos」では少なくともレベル1の**「自動復活（Autorevive）」**または**「ガッツ（Guts）」**をセットすることが必須です。

| ジーン名 | ティア | 主な効果 | 入手方法 |
| :--- | :--- | :--- | :--- |
| 専用ジーン+ (Warpath+) | S | キャラクター固有メカニクスの強化 + サブステータス | フェイトブレイカークエスト |
| 有利属性変換+ (War Elemental+) | S | 弱点属性ダメージを与える + ランダムな防御ステータス | 混成の領域（深層） |
| アルファ V+ (Alpha V+) | A | 奥義ダメージ上限アップ + 奥義ゲージ上昇量アップ | 難易度 Chaos ボス |
| ガンマ V+ (Gamma V+) | A | 奥義発動時にシールド付与 + 攻撃上限アップ | ベルゼバブ・レイド |

トップティアの光属性アタッカーのパワーを最大限に引き出すなど、キャラクター固有のビルドについては、[サンダルフォンビルドガイド](/ja/builds/sandalphon-build/)をご覧ください。公式アップデート、パッチノート、コミュニティのアナウンスについては、[Cygames公式サイト](https://relink.granbluefantasy.jp/en/)をご覧ください。

## よくある質問（FAQ）

### Endless Ragnarok拡張コンテンツを最も早く解放する方法は何ですか？
拡張コンテンツにアクセスするには、『グランブルーファンタジー リリンク』の本編ストーリーをクリアし、クエストカウンターで難易度「Proud」を解放し、クエスト「プロト・バハムトとの戦い」をクリアする必要があります。これがエピローグを解放するための核心的な**Endless Ragnarokの前提クエスト**チェーンとなります。

### 新しい拡張コンテンツはマルチプレイ（コープ）でプレイできますか？
はい。この拡張コンテンツは、PS5、PS4、Switch 2、PC (Steam) 間での完全なクロスプラットフォームプレイ（クロスプレイ）に対応しています。すべての難易度「Chaos」クエスト、フェイトブレイカークエスト、ボスレイドで他のプレイヤーとチームを組むことができます。ただし、**「混成の領域（The Conflux）」**は完全にソロ専用のローグライトモードです。

### 新しいDLCキャラクターの中で、初心者に最もおすすめなのは誰ですか？
初心者にはマギリィが強く推奨されます。彼女の浮遊する武器は安全な距離から安定したダメージを与えることができ、シールドや回復を生成する能力は「混成の領域」のようなソロコンテンツにおいて優れた生存力を提供します。

### 武器の限界突破に必要な素材はどのように入手しますか？
武器の限界突破素材は、主に**「フェイトブレイカークエスト」**のクリアや、**「混成の領域」**の深層をクリアすることで入手できます。最も希少なドロップを得るには、ラグナリアやベルゼバブなどの高レベルのボスを倒す必要があります。
