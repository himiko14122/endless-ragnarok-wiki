究極の**『エンドレス・ラグナロク』戦闘ガイド**へようこそ。『グランブルーファンタジー リリンク』の大型拡張DLCでは、精密な操作が求められる一連の高度な戦闘システムが導入されています。難易度「Chaos（カオス）」のクエストに挑む場合でも、[コンフラックス（The Conflux）](/ja/conflux/the-conflux-complete-guide/)の深部を攻略する場合でも、リンクアタック、奥義（SBA）、そしてチェインバーストのニュアンスをマスターすることは生存に不可欠です。この**『エンドレス・ラグナロク』戦闘ガイド**では、これらの核心となるメカニズムを分解し、新たに導入された召喚システムを分析し、与ダメージを最適化するための実践的な戦略を提供します。

エンドゲームコンテンツを安定してクリアするためには、プレイヤーは単なるボタン連打から脱却しなければなりません。この拡張版では、よりシビアなタイミングウィンドウ、ボス固有のギミック（オーバードライブ/バースト状態の無効化など）、そしてチーム全体の連携リソースの管理が求められます。攻撃の選択肢をどのように連携させるかを理解することで、ボスを永続的な怯み（スタッガー）ループにハメ、最も危険なフェーズを完全にスキップすることができます。

## リンクアタックとリンクタイムのメカニズムをマスターする

リンクアタックは、『エンドレス・ラグナロク』におけるパーティ連携の要です。敵に高威力のスキルや特定のコンボフィニッシュを当てたとき、あるいは特定の隙があるウィンドウ中に攻撃すると、敵のスタンゲージが蓄積されます。スタンゲージが最大になると、青いターゲットプロンプトが表示され、リンクアタックのチャンスを知らせます。 

この**『エンドレス・ラグナロク』リンクアタックガイド**では、UIの中央に表示され、0%から100%まで変化する「リンクゲージ」の重要性を強調します。リンクアタックを実行すると、全体のリンクゲージが15%増加します。リンクゲージが100%に達し、パーティメンバー4人全員が怯んだ敵に対してリンクアタックを成功させると、パーティは「リンクタイム」に突入します。 

リンクタイム中、ボスにとっては戦場の時間の流れが遅くなり、一方でプレイヤーのパーティは強力なバフを獲得します：
*   **時間遅延（スロウ）：** 敵のアクションが50%遅くなります。
*   **スキルクールダウン短縮：** スキルのクールダウン回復速度が75%上昇し、高速なスキル回転が可能になります。
*   **攻撃力＆クリティカルバフ：** 攻撃力が20%増加し、クリティカル確率が100%になります。
*   **HPリジェネ：** すべてのパーティメンバーに持続的なHP回復（毎秒最大HPの1%）が適用されます。

リンクタイムの発生頻度を最大化するために、プレイヤーはスタンダメージとリンクゲージの獲得量を増加させるジーン（Sigil）を装備する必要があります。[ベアトリクス](/ja/characters/beatrix-guide/)のようなキャラクターは、精密な「デルタクロック」のメカニズムを使用して迅速にスタンダメージを与え、このプロセスを加速させることができます。

| アクション | リンクゲージ上昇量 (%) | スタン値倍率 |
| :--- | :--- | :--- |
| 通常リンクアタック | 15% | 基準 (1.0x) |
| ジャスト回避 | 2% | N/A |
| ジャストガード | 2% | N/A |
| 奥義（SBA）発動（自身） | 10% | 1.5x |
| 奥義チェイン参加（仲間） | 5% | 1.0x |

構成を最適化するために、この**『エンドレス・ラグナロク』リンクアタックガイド**の戦略を参考にしてください。スタン値の高いスキルは、ボスが硬直状態（リカバリーロック）のときのために温存しておきましょう。ボスがフェーズ移行中のときは、プロンプトが表示されずにスタンゲージがリセットされてしまうため、リンクアタックを無駄撃ちしないようにしてください。

## 奥義（SBA）とチェインバーストの最適化

奥義（SBA）は、大ダメージを与えると同時に、クエストのタイマーとボスの行動を一時的に停止させるキャラクター固有の必殺技です。難易度「Chaos」において生存率を高めるためには、SBAゲージの管理が極めて重要です。SBAゲージは、攻撃をヒットさせる、ダメージを受ける、あるいはジャスト回避やジャストガードを成功させることで蓄積されます。

この**『エンドレス・ラグナロク』奥義ガイド**では、「奥義チェイン（SBA Chaining）」の概念に焦点を当てます。1人のプレイヤーが奥義を発動すると、カウントダウンタイマーが始まります。タイマーが切れる前に別のプレイヤーが奥義を発動すれば、チェインが継続します。複数の奥義をチェインさせると、シーケンスの最後に「チェインバースト」が発動し、範囲内のすべての敵に強力な属性ダメージを与えます。

参加するパーティメンバーの数に応じて、4つの段階のチェインバーストが存在します：
1.  **2チェイン（Duo Chain）：** 2連続の奥義で発動。
2.  **3チェイン（Trio Chain）：** 3連続の奥義で発動。
3.  **フルチェイン（Full Chain）：** 4連続 of 奥義で発動。
4.  **カタストロフィ（Catastrophe）：** 召喚獣がフィールドにアクティブな状態で、4人のパーティメンバー全員が奥義を使用すると発動する特別なフルチェイン。

| チェインバーストの種類 | ダメージ倍率 | ボーナス効果 |
| :--- | :--- | :--- |
| 2チェイン | 合計奥義ダメージの1.5倍 | 属性防御DOWNを付与 (10% / 15秒) |
| 3チェイン | 合計奥義ダメージの2.2倍 | 属性防御DOWNを付与 (15% / 20秒) |
| フルチェイン | 合計奥義ダメージの3.5倍 | スロウおよび属性防御DOWNを付与 (20% / 30秒) |
| カタストロフィ | 合計奥義ダメージの5.0倍 | 麻痺 (5秒) および属性防御DOWNを付与 (30% / 40秒) |

この**『エンドレス・ラグナロク』チェインバーストガイド**によると、奥義チェインを開始する最適なタイミングは、ボスが「オーバードライブ（Bloodthirst）」モードに入る直前です。ボスのHPゲージが赤く光り、大技の準備に入ったとき、フルチェインを実行することでその移行を遅らせるか完全にキャンセルさせることができ、ボスを通常のDPSローテーションに対して無防備な状態に保つことができます。

## 召喚システムとプライマルバースト

『エンドレス・ラグナロク』では召喚システムが導入され、ルリアが難易度「Chaos」のクエスト中に強力な星晶獣を召喚できるようになります。各プレイヤーは最大4つの召喚獣を編成に装備できますが、戦場に同時に存在できる召喚獣は常に1体のみです。

アクティブな召喚獣は、自律支援型と操作型の2つのカテゴリに分類されます。ルシファーやローランなどの操作型召喚獣では、プレイヤーが直接コマンドを入力して、壊滅的な範囲攻撃や防御バリアを展開することができます。

この拡張版の戦闘ループにおける最も強力な連携要素が、**プライマルバースト（Primal Burst）**です。プライマルバーストを発動するには、パーティは以下の条件を満たす必要があります：
1.  召喚獣がフィールドにアクティブであること。
2.  パーティがリンクタイムを発動させること。
3.  リンクタイムの有効時間内に、4人のパーティメンバー全員が奥義を実行すること。

これらの条件が満たされると、通常のフルチェインがプライマルバーストへとアップグレードされます。この攻撃は通常のダメージ上限を突破し、数百万ポイントの軽減不能なダメージを与え、ベルゼバブやザ・ワールドといったボスに対して長時間のスタン状態を付与します。

| 召喚獣 | アクティブスキル | パッシブパーティボーナス（装備時） | クールダウン |
| :--- | :--- | :--- | :--- |
| **ルシファー** | パラダイス・ロスト (範囲光ダメージ + ディスペル) | スキルダメージ上限 +15% | 180秒 |
| **ローラン** | セイクリッド・シールド (8秒間の無敵バリア) | 防御力 +20% | 150秒 |
| **プロトバハムート** | レギンレイヴ (直線闇ビーム + 裂傷/火傷) | 攻撃力 +10% | 120秒 |
| **ティアマト・マグナ** | テンペストヴェイン (竜巻 + クラウドコントロール) | 奥義ゲージ上昇量 +15% | 90秒 |

召喚獣を効果的に活用するには連携が必要です。プレイヤー間でクールダウンを共有し、激しいボスフェーズ中にはローランのような防御的な召喚獣を使用できるようにし、ルシファーのような攻撃的な召喚獣はリンクタイム中のDPSウィンドウのために温存しておくようにしましょう。

## 戦闘の流れにおけるジーンとキャラクターメカニズムのシナジー

リンクアタックと奥義の効率を最大化するには、キャラクターのビルドが迅速なゲージ生成をサポートしている必要があります。ここでジーン（Sigil）の選択とキャラクターのメカニズムが融合します。[サンダルフォン](/ja/characters/sandalphon-guide/)のような強力なキャラクターは、天司長モード（Supreme Primarch Mode）での高速な多段ヒットコンボに依存しており、他のキャラクターよりも早く奥義ゲージを溜めることができます。

キャラクターをビルドする際の、重要な**『エンドレス・ラグナロク』戦闘のコツ**をいくつか紹介します：
*   **アルファ、ベータ、ガンマジーン：** [武器限界突破（Weapon Transcendence）](/ja/weapon-transcendence/weapon-transcendence-guide/)やフェイトブレイカークエストで入手できるこれらのエンドゲームジーンは、奥義ダメージとゲージ生成に莫大なボーナスをもたらします。
*   **専用ジーン（Warpath+）：** これらのキャラクター専用ジーンは、リンクアタック命中時にクールダウンを短縮するなど、キャラクターの主要なメカニズムにユニークな効果を追加します。
*   **高揚ジーン（Uplift）：** サポートやタンクキャラクターの定番であり、レベル15で奥義ゲージの上昇率を最大30%増加させます。

| ジーンタイプ | セット効果 (2セット) | セット効果 (4セット) | おすすめキャラクター |
| :--- | :--- | :--- | :--- |
| **アルファ (Alpha)** | 奥義ダメージ上限 +10% | リンクアタック使用時に奥義ゲージを10%回復 | サンダルフォン、シャルロッテ、ランスロット |
| **ベータ (Beta)** | リンクタイム中のスキルダメージ上限 +15% | 奥義使用後にスキルのクールダウンを10%短縮 | カリオストロ、イオ、マジェリー |
| **ガンマ (Gamma)** | 奥義発動時に最大HPの10%分のバリアを付与 | チェインバースト後に一時的な攻撃バフ (20%) を付与 | ヴェイン、ジークフリート、ガランツァ |

ビルドを設計する際は、純粋なダメージとユーティリティのバランスを取ってください。「高揚」や「連携攻撃（Linked Together）」ジーンを一切装備せず、純粋なダメージビルドのみを運用するプレイヤーがいると、チーム全体のチェインバーストの回転が遅くなり、長期戦において結果的にチーム全体のDPSが低下することになります。

## 高難易度ボス攻略（Chaosティア）

低レベルのクエストでこれらのメカニズムを実行するのは簡単ですが、ベルゼバブ、ザ・ワールド、ラグナリアといった難易度「Chaos」のエンカウントでは、精密な立ち回りが求められます。プレイヤーはクリアするために、体系化された戦闘の流れに従う必要があります。これらの戦闘におけるキャラクタービルドの詳細については、[Cygames公式のグランブルーファンタジー リリンク ポータル](https://relink.granbluefantasy.jp/en/)をチェックしてください。

高難易度ボスに対する戦闘の流れを管理するための、段階的な戦略は以下の通りです：

1.  **フェーズ1：ゲージ溜め (ボスHP 100% - 70%)**
    *   コンボフィニッシュやジャスト回避を当てることに集中し、奥義ゲージとリンクゲージの両方を溜めます。
    *   奥義を単発で使用しないでください。パーティメンバー4人全員が100%に達するまで温存します。
2.  **フェーズ2：スタッガーロック (ボスHP 70%)**
    *   ボスが最初の主要なギミック（大技）の準備に入ったら、奥義チェインを開始します。
    *   「ディレイチェイン（スローチェイン）」を使用します。発動中の奥義のカウントダウンタイマーがゼロになる直前まで待ってから次の奥義を発動します。これにより、ボスがフリーズする時間を引き延ばすことができます。
3.  **フェーズ3：リンクタイム＆召喚フェーズ**
    *   フルチェインが発動し、ボスは怯み状態になります。すぐにリンクアタックを実行してリンクゲージを100%にし、リンクタイムを発動させます。
    *   ルリアはすぐにルシファーまたはローランを召喚します。
    *   リンクタイム中は、最もクールダウンの短いスキルを連打してダメージを最大化します。
4.  **フェーズ4：プライマルバーストの発動**
    *   リンクタイムが終了する直前に、2回目の奥義（アルファジーンのセット効果や迅速なスキル使用により、ある程度リチャージされているはずです）を実行します。
    *   これによりプライマルバーストが発動し、大ダメージを与えてボスの防御フェーズを完全にスキップします。

このローテーションを遵守することで、チームはラグナリアやベルゼバブ戦の最も危険なフェーズをスキップし、安定したクリアを達成できます。

## エンドレス・ラグナロク戦闘FAQ

### リンクゲージを早く溜めるにはどうすればよいですか？
リンクゲージを素早く溜めるには、最大レベルでリンクゲージ獲得量を最大25%増加させる**「連携攻撃（Linked Together）」**ジーンを装備してください。さらに、ジャスト回避やジャストガードを成功させ、プロンプトが表示されたら必ずリンクアタックを行うことで、ゲージが加速します。一部のキャラクターはスタン値を高める特定のスキルを持っており、これによりリンクアタックのプロンプトがより頻繁に発生するようになります。

### 2人だけでチェインバーストを発動することはできますか？
はい、チェインバーストは最低2人のプレイヤー（2チェイン）から発動可能です。ただし、ダメージ倍率やボーナス効果は参加人数に応じてスケールします。終盤のボス戦では、ダメージを最大化し、クラウドコントロール効果を付与するために、フルチェイン（4人）またはプライマルバースト（4人＋アクティブな召喚獣）を強く推奨します。

### フルチェインとプライマルバーストの違いは何ですか？
フルチェインは、通常の条件下で4連続の奥義を実行することで発動します。一方、**プライマルバースト**は、*召喚獣*がフィールドにアクティブな状態で、*リンクタイム中*に4つの奥義をすべてチェインさせたときに発生するアップグレード版です。プライマルバーストは大幅に高いダメージを与え、通常のダメージ上限を突破し、対象に固有の「麻痺」効果を付与します。

### 召喚獣はフィールドに永続的に残りますか？
いいえ。召喚獣のアクティブ時間は限られており（通常、召喚獣に応じて30〜45秒）、またはアクティブスキルが完全に解決されると消失します。また、クールダウンも長いため（90〜180秒）、重要なDPSウィンドウや決定的な防御フェーズのために温存しておく必要があります。

## FAQ

**Q: マスターすべき最も重要な戦闘メカニズムは何ですか？**
A: 回避のタイミングが最も重要なメカニズムです。ボスの予備動作アニメーションを学習し、適切なタイミングで回避することで、被ダメージを防ぎつつ奥義ゲージを溜め続けることができます。難易度「Chaos」では、一度の回避ミスがキャラクターの戦闘不能につながる可能性があります。

**Q: マルチプレイ（コープ）における奥義チェインシステムはどのように機能しますか？**
A: 4人のプレイヤーが順番に奥義を実行すると、大ダメージを与えてリンクタイムを延長させるフルチェインが発動します。重要なのはコミュニケーションです。定型文チャットを使用して「奥義準備完了！」と合図を送り、順番に実行しましょう。召喚獣がアクティブなリンクタイム中にフルチェインを行うと、さらに高いダメージを与えるプライマルバーストになります。

**Q: リンクタイムと通常の戦闘の違いは何ですか？**
A: リンクタイムは、リンクアタックの成功後に発動する一時的な状態です。リンクタイム中、すべてのパーティメンバーの与ダメージが増加し、スキルのクールダウンが速くなり、パーティ全体がノックバックに対して無効になります。これは強敵に対する主要なDPSウィンドウとなります。

**Q: 戦闘での与ダメージを最大化するにはどうすればよいですか？**
A: スキルローテーションを常に維持することに集中し、バーストスキルをリンクタイムのウィンドウのために温存し、適切な属性有利を活用し、フルチェインのために奥義のタイミングを合わせることに注力してください。限界突破（Master Trait）やジーンの最適化をマスターすることで、ダメージ上限をさらに引き上げることができます。
