「The Conflux（コンフラックス）」は、『グランブルーファンタジー リリンク』の拡張DLC『Endless Ragnarok』で導入された、最高峰のローグライトソロモードです。個人の戦闘技術を試すために設計されたこのモードでは、AIのパーティーメンバーは排除され、プレイヤーは操作キャラクター1人、ランダムなパワーアップ、および自身の操作技術のみを頼りに戦うことになります。変化し続ける深部を進んでいくことは、[武器限界超越（Weapon Transcendence）](/ja/weapon-transcendence/weapon-transcendence-guide/)に必要な希少素材を集め、高ランクの[マスター特性（Master Traits）](/ja/master-traits/master-traits-explained/)を解放するための最速の方法です。

初めての挑戦を生き抜くための**Endless Ragnarok Confluxガイド**を探している方も、周回効率を最適化するための高度な戦略を求めている方も、この包括的な解説を読めば、この高難度ゲームモード内のすべてのギミック、部屋のタイプ、報酬システムを把握できます。

## The Confluxの解放条件

このローグライトモードにアクセスするには、まず本編の難易度Proudクエスト「終局の刻、交わされる約束（The Tale of Bahamut's Rage）」をクリアする必要があります。このクエストをクリアすると、『Endless Ragnarok』のエピローグストーリーが発生します。拡張コンテンツの導入カットシーンを視聴すると、「Conflux Gateway（コンフラックスの門）」が恒久的に活性化します。

これらのゲートウェイは、2つの主要な拠点にある物理的な建造物です。
1. **辺境の街シードホルム (Seedhollow):** クエストカウンター近くの西の広場にあります。
2. **トレダメ聖域 (Tredame Sanctum):** 中央の祭壇の近くに配置されています。

どちらかのゲートウェイを調べることでConfluxのインターフェースが開き、使用するキャラクターを選択して挑戦を開始できます。これは完全なソロモードであるため、パーティー編成は無視され、操作キャラクターの装備、ジーン、解放済みのマスタリーのみが挑戦に引き継がれます。

## 基本ループと部屋の仕組み

The Confluxの通常の挑戦は、一般に「階層（Depths）」と呼ばれる進行型のフロアで構成されています。最後のボス部屋まで生存できた場合、1回の挑戦にかかる時間は約20分です。部屋のレイアウトはランダムに生成され、各エリアをクリアした後に分岐ルートが提示されます。

```
[ゲートウェイ入場] ➔ [初期の祝福を選択]
                       │
                       ▼
           ┌───────────────────────┐
           │     次の階層を選択     │
           └───────────┬───────────┘
                       │
         ┌─────────────┼─────────────┐
         ▼             ▼             ▼
  [戦闘部屋]     [宝箱部屋]    [レアモンスター]
         │             │             │
         └─────────────┼─────────────┘
                       │
                       ▼
             [祝福の強化を選択]
                       │
                       ▼
             [ボス部屋 (階層 10/20/30)]
```

部屋をクリアすると、次の階層へと続く2〜3個のゲートウェイポータルが表示されます。各ポータルには、部屋のタイプと、クリア時に獲得できる大まかな報酬のカテゴリが表示されます。

### 部屋のタイプ

| 部屋のタイプ | 目標 | 危険度 | 主な報酬 |
| :--- | :--- | :--- | :--- |
| **戦闘部屋** | 制限時間内に通常のモンスターの群れを撃破する。 | 低〜中 | 下位の祝福、武器強化素材 |
| **宝箱部屋** | 環境トラップ（トゲ、落石など）を避けながら宝箱を開ける。 | 低 | Confluxトークン、ジーン（A〜Sランク） |
| **レアモンスター部屋** | 強力なエリート敵（ラグナリア・ヘラルドやパイエット・Aの変異体など）を1体撃破する。 | 高 | 上位の祝福、超越の残り火（Transcendence Embers） |
| **休息のオアシス** | 非戦闘部屋。HPの回復、装備ジーンの入れ替え、または一時的なバフの購入が可能。 | なし | HP回復、ジーン調整 |
| **ボス部屋** | 階層10、20、30で主要なボス（ザ・ワールドやベルゼバブの幻影など）を撃破する。 | 極めて高 | フェイトブレイカーの断片、マスター特性ポイント |

適切なルート選択が極めて重要です。キャラクターの残りHPが少ない場合は、無理にレアモンスター戦に挑むよりも、休息のオアシスや宝箱部屋を優先する方が安全です。ただし、難易度の高い部屋を避けると、集まる一時的なパワーアップの質が下がり、最終ボス戦が大幅に難しくなります。

## 祝福システム：一時的なパワーアップ

各階層に入る前に、**「祝福（Blessings）」**と呼ばれるランダムに提示される3つのパワーアップから1つを選択する必要があります。これらのバフは挑戦中のみ有効で、退出するか失敗すると消失します。特に複雑なメカニクスを持つキャラクターを使用する場合、祝福のシナジーを考慮した戦略が攻略の鍵となります。

祝福は、ゲームの育成システムに対応する3つの系統に分類されます。
* **攻撃の祝福（赤）:** 攻撃力、クリティカルダメージ、アビリティのクールダウン速度、追撃ダメージを強化します。
* **防御の祝福（青）:** 最大HPの増加、自動復活の付与、シールドの追加、または攻撃時のHP吸収効果を付与します。
* **支援の祝福（黄）:** リンクタイムの発生率向上、奥義（SBA）ゲージの上昇速度アップ、またはConfluxトークンのドロップ量を増加させます。

### 高階層攻略におすすめの祝福

| 祝福名 | レアリティ | 効果 | 相性の良いキャラクター |
| :--- | :--- | :--- | :--- |
| **ヴェスパーズエッジ** | レジェンダリー | HPが80%以上のとき、クリティカルダメージが45%増加する。 | サンダルフォン、シャルロッテ |
| **クロノスアクセラレーション** | レア | ジャスト回避成功時、アビリティのクールダウンを15%短縮する。 | ランスロット、ヨダルラーハ |
| **パクトベアラーズイージス** | レジェンダリー | 最大HPの50%分のシールドを付与。30秒ごとに再生成される。 | フラウ、ヴェイン |
| **シャッターストライク** | コモン | 強攻撃時のスタン値が25%増加する。 | バザラガ、ジークフリート |
| **エーテルフロウ** | レア | リンクタイム中、奥義ゲージの上昇速度が20%アップする。 | イオ、カリオストロ |

祝福を選ぶ際は、乗算バフを重ねることを意識しましょう。例えば、「ヴェスパーズエッジ」とクリティカル確率アップを組み合わせることで、ランスロットのような手数の多いアタッカーを圧倒的なアタッカーへと変貌させることができます。[ベアトリス（Beatrix）](/ja/characters/beatrix-guide/)を使用している場合は、彼女のハイリスクな「デルタクロック」メカニクスとシナジーを生むために、低HP時にダメージが上昇するマルチプライヤーを探すと良いでしょう。

## The Confluxにおけるキャラクター最強ランキングと戦略

28人のプレイアブルキャラクター全員が、このソロ環境で同等のパフォーマンスを発揮できるわけではありません。回復や敵のヘイトを引いてくれるサポートキャラクターがいないため、自己完結能力、機動力、および瞬間火力が最も重視される特性となります。

### SSランク：サンダルフォン
サンダルフォンは、**「天司長（Supreme Primarch）モード」**により、The Confluxにおいて圧倒的な強さを誇ります。軽快なステップから剣気波を連発できる能力により、高い機動力を維持しながら安全な距離から大ダメージを与えることができます。さらに、自己回復スキルを標準搭載しているため、回復薬が制限される深部への挑戦において驚異的な生存力を発揮します。

### Sランク：カリオストロ ＆ バザラガ
* **カリオストロ:** 優れたユーティリティ性能を持ちます。コラプス攻撃は優秀な範囲ダメージを与え、自己復活や回復能力により、ミスをしても挑戦が即座に終了するのを防ぐことができます。
* **バザラガ:** 究極のタンクです。タメ攻撃によって一時的な無敵フレームと大幅な被ダメージカットを得られるため、部屋のトラップの大部分を無視し、エリート敵との戦闘をごり押しで突破できます。

### Aランク：ベアトリス ＆ マギエル（DLCキャラクター）
* **ベアトリス:** ハイリスク・ハイリターン。デルタクロックのメカニクスにより爆発的なダメージを出せますが、HP管理には正確な回避が求められます。
* **マギエル:** 非常に優れた防御オプションです。展開されたシールドで身を守りながら、浮遊する武器を召喚して敵を攻撃できるため、生存に苦戦しているプレイヤーにとって安全な選択肢となります。

ソロコンテンツ向けにこれらのキャラクターをビルドする詳細な方法については、[Endless Ragnarok キャラクターガイド](/ja/characters/)をご覧ください。

## The Confluxにおける武器限界超越素材の周回

プレイヤーがこのローグライトモードに何時間も費やす主な理由は、**「武器限界超越（Weapon Transcendence）」**の素材を集めるためです。このエンドゲームシステムでは、最大まで強化した武器（守護、昇華、処刑、会心、気絶、および究極）を初期限界を超えて強化し、付け替え可能な特性やユニークな見た目のエフェクトを解放できます。

武器を超越させるには、The Confluxの高階層ボス、または高難度のフェイトブレイカー（Fatebreaker）クエストからのみドロップする特定の素材が必要です。

### 階層別の超越素材ドロップ率

| 階層範囲 | 主なボス戦 | 主なドロップ素材 | ドロップ確率 |
| :--- | :--- | :--- | :--- |
| **階層 1〜10** | パイエット・Aの幻影 | 超越の残り火（Transcendence Ember） | 100% (1〜2個) |
| **階層 11〜20** | ラグナリア・ヘラルド | 精製アストラル鉱石（Refined Astral Ore） | 75% (1個) |
| **階層 21〜30** | ザ・ワールド（アーカルム） | フェイトブレイカーの断片（Fatebreaker Fragment） | 50% (1個) |
| **階層 31以上** | ベルゼバブの幻影 | ケイオスコア / フェイトブレイカーの断片 | 100% (どちらか1個) |

周回効率を最適化するには、ゼタやランスロットなど、移動速度が速く雑魚処理能力の高いキャラクターを使用することをお勧めします。究極武器（Terminus weapons）の強化において最大のボトルネックとなる「フェイトブレイカーの断片」を確保するために、少なくとも階層30への到達を目指しましょう。

## 生存のためのConflux高度なコツ

序盤の階層を進むのは比較的簡単ですが、階層25を超えると難易度が劇的に上昇し、ゲームの悪名高き**「Chaos（混沌）難易度」**に匹敵するようになります。一度のミスが即死につながることもあります。挑戦を生き延びるために、以下の高度な戦術を活用してください。

1. **キャラクター専用ジーンの装備:** 挑戦を開始する前に、キャラクターにSランクの「覇道+（Warpath+）」やキャラクター専用ジーンが装備されていることを確認してください。これらのユニークな特性はConflux内にも引き継がれ、通常の「祝福」では再現できない強力な効果を発揮します。
2. **リンクタイムの活用:** ソロプレイでは、ジャスト回避、ジャストガード、およびスタン発生後の追撃を行うことでリンクゲージが上昇します。リンクタイムはエリート戦やボスの特定のフェーズまで温存し、敵の動きを止めて一方的にダメージを与えるために使いましょう。
3. **召喚獣のパッシブボーナスの利用:** 挑戦の序盤ではルシファーやローランなどの召喚獣をアクティブに操作することはできませんが、高ランクの召喚獣を装備しておくことで、祝福の効果と乗算されるパッシブステータス上昇効果を得られます。ゲートウェイに入る前に、召喚獣の編成が最適化されているか確認してください。
4. **クールダウン短縮の優先:** アビリティを発動できなければ、どれだけ攻撃力が高くても意味がありません。挑戦の序盤で、少なくとも1つは上位のクールダウン短縮の祝福を選択し、防御アビリティや攻撃アビリティを常に使用できる状態にしておきましょう。

ゲートウェイに入る前の装備編成の最適化に関する詳細情報は、パッチアップデートやバランス調整が掲載されている公式の[Cygames Granblue Fantasy: Relink Portal](https://relink.granbluefantasy.jp/en/)をご参照ください。

## よくある質問

### The Confluxのゲートウェイを解放するにはどうすればよいですか？
本編の難易度Proudクエスト「終局の刻、交わされる約束」をクリアする必要があります。これにより『Endless Ragnarok』のエピローグが解放されます。最初のエピローグカットシーンを進めると、シードホルムとトレダメ聖域にあるゲートウェイが開きます。

### マルチプレイ（協力プレイ）でThe Confluxをプレイすることはできますか？
いいえ。The Confluxは完全なソロ専用ローグライトモードです。他のプレイヤーやAI制御のパーティーメンバーを同行させることはできません。選択したキャラクター1人のみで部屋をクリアする必要があります。

### Confluxの挑戦中に死亡するとどうなりますか？
死亡し、自動復活や回復アイテムが尽きると、その時点で挑戦は即座に終了します。それまでに集めたConfluxトークンや基本的な素材は維持されますが、階層30以上の最終ボスを撃破した際に得られるボーナス完了報酬は失われます。

### Confluxの素材を使って、最初にどの武器種を超越させるべきですか？
最初に**究極武器（Terminus Weapons）**に集中することを強くお勧めします。これらの武器はゲーム内で最も高いダメージ上昇率を誇り、今後のConfluxの周回や高難度のフェイトブレイカークエストの攻略が大幅に楽になります。
