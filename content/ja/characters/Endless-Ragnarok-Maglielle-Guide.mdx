「グランブルーファンタジー リリンク」の追加DLC「エンドレス・ラグナロク」にて、マグニエールが屈指の防御性能を誇るキャラクターとして参戦します。遠距離サポートキャラクターとして、彼女は高効果のバリア、範囲回復、そして安全な位置から継続的なダメージを与える自動飛翔武器により、チームの生存率を維持することに長けています。高難易度のエンドゲームコンテンツにおいて、**エンドレス・ラグナロクのマグニエール**のセットアップをマスターすれば、本来なら致命的となるボスのギミックを無効化することができます。

[混沌の渦（The Conflux）](/ja/conflux/)のソロモードの深部に挑む場合でも、難易度「カオス」のクエストで[ベルゼバブ](/ja/bosses/beelzebub-boss-guide/)と対峙する場合でも、マグニエールは比類なきユーティリティを提供します。彼女のスキル構成は、防御呪文を唱えている間も自動で攻撃する幻影の刃を召喚することを中心に展開します。この**エンドレス・ラグナロク マグニエール サポート**ガイドでは、彼女のコアメカニクス、最適なビルド、スキルパス、および現在のキャラクターメタにおける彼女の位置づけについて解説します。

## コアメカニクスとアウレオラシステム

マグニエールの戦闘ループは、スキルショートカットの上に表示される独自のリソースバーである**アウレオラゲージ**を中心に展開します。このゲージは5つのセグメントに分かれており、それぞれが浮遊する幻影の刃を表しています。基本攻撃コンボや特定のスキルを実行することで、マグニエールはこれらの刃を召喚して自身の周囲に浮遊させ、範囲内の敵を自動的に攻撃させます。

```
[ アウレオラゲージ: ▢ ▢ ▢ ▢ ▢ ] -> アクティブな刃: 最大5本
  - 弱攻撃コンボ: 4段目のヒットで刃を1本生成
  - 強攻撃（チャージ）: すべてのアクティブな刃を発射してバーストダメージを与える
  - サポートスキル: アクティブな刃を消費して回復とシールドを強化する
```

### 基本攻撃の入力とゲージ生成

ダメージとユーティリティを最大化するには、アウレオラの刃を素早く生成し、消費する方法を理解する必要があります。マグニエールの弱攻撃コンボは、魔法の弾丸を連射するものです。4段目の基本コンボの最後のヒットで刃が1本召喚されます。逆に、強攻撃は現在召喚されているすべての刃を扇状に貫通発射し、消費した刃の数に応じてスケールするダメージを与えます。

| 入力コマンド | アクション名 | アウレオラゲージ効果 | 説明 |
| :--- | :--- | :--- | :--- |
| **弱攻撃（コンボ4回）** | スペクトラルニードル | 刃を+1生成 | 遠距離魔法弾コンボ。最後のヒットで浮遊する短剣を召喚。 |
| **強攻撃（タップ）** | ブレードリリース | すべての刃を消費 | アクティブな刃を前方に発射。10%の防御力ダウンのデバフを付与。 |
| **強攻撃（長押し）** | アウレオラサルヴォ | すべての刃を消費 | 貫通ビームをチャージ。刃が5本の時にダメージが150%増加。 |
| **空中弱攻撃** | フォーリングシャード | なし | 素早い下方向への弾丸。位置調整に有効。 |
| **リンクアタック** | ハーモニックストライク | 刃を+3生成 | 瞬時にゲージを溜め、連携パーティ攻撃を発動。 |

### スキルのシナジー：回復とシールド

マグニエールのアクティブスキルは、アクティブなアウレオラの刃の数に応じて特性が変化します。**エンドレス・ラグナロクのマグニエールの回復**呪文や防御バリアを唱える際、ゲージが最大であると、効果の範囲、持続時間、および効力が増加します。

*   **イージスウォール**: 静止型のドーム状シールドを展開します。基本レベルでは飛び道具をブロックします。刃が5本の状態でキャストすると、ドーム内のすべての味方に5秒間の**無敵**効果を付与します。
*   **セイクリッドサンクチュアリ**: 地面に回復エリアを設置します。1.5秒ごとに最大HPの一定割合を回復します。刃を消費することで、回復の発生間隔が短縮され、エリアの持続時間が延長されます。
*   **ヴァージドライブ**: すべての刃を消費して巨大な回転剣の渦を放つ、高ダメージの攻撃スキルです。これは敵の部位破壊やボスを怯ませるための彼女の主な手段です。

## マスタートレイトとスキルツリーのルート

レベル100に達すると、マグニエールはマスタートレイトにアクセスできるようになります。これらのトレイトは、**インサイト（Insight）**、**エッセンス（Essence）**、**クリュクス（Crux）**の3つの異なる進行ルートに分かれています。これらのノードを解放するには、レベルアップ進行、フェイトブレイカー（Fatebreaker）クエスト、および[混沌の渦（The Conflux）](/ja/conflux/)のクリアで獲得できるマスターポイント（MSP）が必要です。

### 3つのマスタートレイトルート

1.  **インサイトルート（サポート＆回復特化）**: このルートはマグニエールの防御ユーティリティを強化します。バリアスキルのクールダウンを短縮し、範囲回復呪文の回復上限を増加させ、主要な回復スキルに弱体効果解除（クリア）効果を追加します。
2.  **エッセンスルート（DPS＆刃の自動化）**: このルートは彼女の飛翔武器に焦点を当てています。刃の攻撃速度を上げ、クリティカル率を高め、強攻撃を放った際に確率で刃を払い戻す効果を付与します。
3.  **クリュクスルート（ユーティリティ＆リンクタイム）**: このルートはパーティ全体のユーティリティを向上させます。奥義（SBA）ゲージの獲得率を増加させ、リンクタイム中のリンクテンペストを強化し、マグニエールに高ランクのクラウドコントロール能力を付与します。

| マスタートレイトルート | ランク特典1 (レベル1) | ランク特典2 (レベル5) | ランクスタイルアップグレード (レベル10) |
| :--- | :--- | :--- | :--- |
| **インサイト** | 回復上限 +10% | セイクリッドサンクチュアリに弱体効果解除を追加 | イージスウォールがすべての味方に怯み無効（Stout Heart）を付与 |
| **エッセンス** | 刃の自動攻撃ダメージ +15% | 刃の払い戻し確率 20% | アウレオラサルヴォが敵の防御力を15%無視 |
| **クリュクス** | 奥義ゲージ上昇量 +8% | リンクアタックダメージ +15% | リンクタイムがアクティブな刃の持続時間を8秒延長 |

**エンドレス・ラグナロクのマグニエール サポート**能力を最適化するには、まずインサイトルートをランクスタイルアップグレードまで優先して解放してください。これにより、[ザ・ワールド](/ja/bosses/the-world-boss-guide/)戦などの難易度「カオス」の遭遇戦における大ダメージフェーズをチームが生き延びられるようになります。

## エンドレス・ラグナロク マグニエールの最適ビルド

効果的な**エンドレス・ラグナロクのマグニエール ビルド**を作成するには、彼女の攻撃的な飛翔武器とサポート能力のバランスを取る必要があります。以下は、エンドゲームの進行向けに高度に最適化された2つのビルドです。

### ビルド1：オーラム・イージス（純サポート＆生存特化）

このビルドは、生存が最優先される高難易度のマルチプレイヤーロビーやカオスクエストに挑むために設計されています。バリアの稼働時間を最大化し、スキルのクールダウンを短縮し、ボスのフェーズ移行中もチームの体力を維持することに焦点を当てています。

*   **武器**: *覚醒武器（限界突破済）* — 高いHPとクールダウン短縮ボーナスを提供します。
*   **主要スキル**: イージスウォール、セイクリッドサンクチュアリ、ディバイングレース、ヴァージドライブ。
*   **優先ジーン**:
    *   **アウレオラの慈愛+**（キャラクター専用）: 飛翔武器が敵にヒットするたびにスキルのクールダウンを2%短縮。
    *   **ウォーパス+**（Sティア）: 自身のバリアエリア内にいる間、防御力が20%増加し、デバフ無効を付与。
    *   **怒涛 V+** ＆ **クイックアビリティ V+**: イージスウォールのクールダウンを維持するために不可欠。
    *   **万能薬 V+**: 自動復活や追加の回復による自己維持能力を向上。

### ビルド2：フライング・アーセナル（ハイブリッドDPS＆サポート）

混沌の渦などのソロコンテンツを素早くクリアしたいプレイヤー向けに、このハイブリッドビルドはマグニエールの飛翔剣のダメージを最大化しつつ、緊急時のセーフティネットとしてサポートスキルを維持します。

*   **武器**: *究極武器（限界突破済）* — 最高の素の攻撃力とクリティカルダメージのスケールを提供します。
*   **主要スキル**: ヴァージドライブ、アウレオラレイン、イージスウォール、ブレードラッシュ。
*   **優先ジーン**:
    *   **アウレオラの覇気+**（キャラクター専用）: 飛翔武器のダメージ上限を30%増加。
    *   **有利属性変換+**: すべての攻撃が敵の弱点属性になります。
    *   **アルファ V+** ＆ **ガンマ V+**: 奥義ダメージを増加させ、奥義発動時にシールドを付与。
    *   **クリティカル確率 V+**: 自動攻撃の刃によるクリティカルヒットを安定させます。

```
ハイブリッドビルドの優先ジーン:
[有利属性変換+] -> [アウレオラの覇気+] -> [クリティカル確率 V+] -> [アルファ/ガンマ V+]
```

| ジーン名 | ティア | 主な効果 | サブ効果 (V+の場合) |
| :--- | :--- | :--- | :--- |
| **アウレオラの慈愛** | S (キャラクター専用) | 刃のヒットごとにスキルCD -2% | 回復上限 +10% |
| **アウレオラの覇気** | S (キャラクター専用) | 飛翔武器のダメージ上限 +30% | クリティカルダメージ +15% |
| **有利属性変換+** | S (ユニーク) | 攻撃が常に弱点属性になる | 攻撃力 +5% |

| **アルファ V+** | A | 奥義ダメージ上限を増加 | 味方の奥義ゲージを上昇 |

## サモンシステムとの連携と戦闘戦略

この拡張版では刷新されたサモンシステムが導入され、プレイヤーは難易度「カオス」のクエストで強力な星晶獣を召喚できるようになります。マグニエールは、素早い奥義ゲージ生成能力と、召喚キャストモーション中にパーティを保護する能力により、サモン特化の戦略において極めて重要な役割を果たします。

### プライマルバーストの発動

エンドレス・ラグナロクでは、パーティが十分なリンクエネルギーを溜めると、ルリアが星晶獣を召喚できます。リンクタイム中にパーティメンバー4人全員が奥義（SBA）を発動すると、**プライマルバースト**がトリガーされ、ルシファーやローランといった操作可能なエンティティが召喚されます。

```
召喚発動の戦闘ループ:
1. アウレオラゲージを溜める -> 2. アウレオラレインで奥義ゲージを溜める -> 3. リンクタイムを発動 -> 4. 4チェイン奥義を発動 -> 5. ルシファー / ローランを召喚
```

マグニエールでこれを効果的に実行するには：
1.  **刃の稼働時間を維持する**: 常に5本の刃をアクティブにしておきます。自動攻撃により、ボスの攻撃を回避している間も奥義ゲージが生成されます。
2.  **チェイン前にイージスウォールをキャストする**: 奥義チェインを開始する直前にイージスウォールを展開します。無敵バフはチェインのアニメーション中も維持され、ボスの流れ弾から味方を守ります。
3.  **ポジショニング**: 中距離を維持します。マグニエールの弾丸には弾速（到達時間）がありますが、離れすぎると回復エリアの効果が薄れてしまいます。

サモンのアップグレードやパッシブボーナスの最大化に関する詳細な情報は、総合的な[サモンシステムガイド](/ja/summons/summon-system-guide/)を参照してください。

## 武器の限界突破（限界超越）とエンドゲームの進行

マグニエールを限界まで強化するには、[武器の限界突破（限界超越）](/ja/weapon-transcendence/)システムを利用する必要があります。このメカニクスにより、完全に強化された武器を通常のレベル上限を超えてアップグレードし、入れ替え可能なトレイトスロットやユニークなビジュアルエフェクトを解放できます。

マグニエールの武器を限界突破させるには、フェイトブレイカークエストや混沌の渦の深部周回から素材を集める必要があります。

### 限界突破素材と優先度

武器のアップグレードには特定のボスドロップ素材が必要です。マグニエールの場合、ダメージ上限を最大化するために、彼女の究極武器である*クラウ・ソラス（Claidheamh Soluis）*のアップグレードを最優先します。

| 限界突破レベル | 必要素材 | ルピ（ゴールド） | 解放されるトレイトスロット |
| :--- | :--- | :--- | :--- |
| **レベル1** | フェイトブレイカーの破片 x10、混沌の残り火 x5 | 150,000 | スロットII (最大レベル15) |
| **レベル2** | アストラルの羽 x15、ベルゼバブの羽 x2 | 250,000 | スロットII (最大レベル25) |
| **レベル3** | 原初の涙 x5、オメガコア x1 | 400,000 | スロットIV (最大レベル5) |
| **レベル4** | フェイトブレイカーの破片 x20、ラグナリアの角 x3 | 600,000 | スロットIV (最大レベル10) |

スロットIIには、サポート能力を向上させるために**イージス維持（Aegis Upkeep）**または**怯み無効（Stout Heart）**を狙いましょう。スロットIVには、エンドゲームの戦闘で飛翔武器が最大ダメージを与えられるよう、**ダメージ上限ブースト（Damage Cap Boost）**に焦点を当てます。

## ティアリストの立ち位置と相性

現在の**エンドレス・ラグナロク マグニエール ティア**評価において、彼女は**Aティア**（特にサポートとユーティリティ部門）に位置しています。[サンダルフォン](/ja/characters/sandalphon-guide/)（SSティア）のようなゲームバランスを崩すほどの純粋なDPSには及びませんが、高難易度コンテンツを安定してクリアするための彼女のユーティリティは不可欠です。

### 相性分析

*   **ベルゼバブ（有利）**: マグニエールのイージスウォールは、ベルゼバブの画面全体の黒い波動攻撃を完全に無効化します。また、遠距離のプレイスタイルにより、彼の近接なぎ払い攻撃から安全な距離を保つことができます。
*   **ザ・ワールド（普通）**: ザ・ワールドは頻繁にテレポートするため、マグニエールの自動攻撃の刃が安定してヒットし続けるのが困難です。しかし、混沌としたフェーズ移行期において、彼女の回復エリアは極めて重要です。
*   **シエテ＆ソーン（有利）**: マグニエールはシールドを展開してソーンのチャージ矢の弾幕をブロックし、近接DPSキャラクターが中断されることなく攻撃できるようにします。

他の5人のDLCキャラクターとの比較など、より広い視野での評価については、[エンドレス・ラグナロク キャラクターティアリスト](/ja/tier-list/)をご覧ください。また、ゲームのアップデートやパッチノートの詳細は、[Cygames公式アップデートポータル](https://relink.granbluefantasy.jp/)で直接確認できます。

## よくある質問

### エンドレス・ラグナロクでマグニエールを解放するにはどうすればよいですか？
マグニエールを解放するには、まず難易度「PROUD」のクエスト「バハムート・オルタの怒り」をクリアしてエピローグコンテンツを解放する必要があります。解放後、フェイトブレイカークエストで入手したキャラクター解放チケットを使用するか、ストアから直接DLCパックを購入することで仲間に加えることができます。

### マグニエールはメインDPSとして活躍できますか？
限界突破した究極武器とジーン「アウレオラの覇気」を装備すれば、マグニエールはかなりのダメージを出すことができますが、彼女のスキル構成は基本的にサポート向けに設計されています。純粋なアタッカーとしてではなく、DPSとサポートのハイブリッドとして運用するのが最適です。

### マグニエールの回復量は攻撃力に比例しますか？
はい、**エンドレス・ラグナロクのマグニエールの回復**スキルは、全体の攻撃力の一定割合に応じてスケールし、特定の回復上限まで増加します。エンドゲームコンテンツで回復能力を最大化するには、「回復上限 V+」などのジーンを装備する必要があります。

### マグニエールと組み合わせるのに最適なサモンは何ですか？
マグニエールと組み合わせるのに最適なサモンはローランです。ローランのパッシブボーナスは、アクティブなバリアエリア内に立っているすべてのパーティメンバーの防御力と回復速度を向上させるため、マグニエールのイージスウォールと直接シナジーを発揮します。
