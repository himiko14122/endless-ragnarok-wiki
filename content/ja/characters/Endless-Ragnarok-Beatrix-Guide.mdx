ベアトリクスは、『グランブルーファンタジー リリンク』の拡張DLC「Endless Ragnarok」にて、ハイリスク・ハイリターンな火属性DPSキャラクターとして参戦しました。彼女の代名詞である封印武器「エムブラスク」を手に、正確なタイミングとアクティブなバフ管理が求められる攻撃的な近接戦闘を得意としています。このEndless Ragnarokベアトリクス攻略ガイドでは、複雑な「デルタクロック」システムのマスター方法、最適なアビリティの分析、そして難易度「Chaos」のクエストを攻略するためのエンドゲーム向け最強ビルドについて詳しく解説します。

他の5人のDLCキャラクターと同様に、Endless Ragnarokにおけるベアトリクスのティア評価に関するコミュニティの合意はまだ形成途中ですが、彼女の圧倒的な最大火力はバザラガやゼタといったトップクラスのDPSキャラクターと十分に渡り合えるポテンシャルを秘めています。この最大出力を引き出すには、ボタンを連打するのではなく、デルタクロックのフェイズを能動的にコントロールする必要があります。自己バフのタイミングを理解し、リンクタイムと合わせることで、ベアトリクスはベルゼバブやザ・ワールドといったエンドゲームのボスたちのHPゲージを一瞬で削り取ることができます。

## デルタクロックの仕組みを解説

ベアトリクスを使いこなす上での核心は、HPバーの上に表示される独自のUIゲージ「デルタクロック」の管理にあります。デルタクロックは、**ネイル（攻撃）**、**オレスト（防御）**、**ゼファー（回復）**という3つの異なるフェイズを常に循環しています。各フェイズはデフォルトで8秒間持続し、その後時計回りに次のフェイズへと自動的に移行します。

```
   [ ネイル (赤) ]  -->  攻撃力 +30%, ダメージ上限 +15%
          |
          v
   [ オレスト (青) ] -->  防御力 +40%, 怯み無効
          |
          v
  [ ゼファー (黄) ]-->  HP吸収 3%, アビリティ時短 +20%
```

ベアトリクスを最適に運用するには、特定のコンボフィニッシュやアビリティを使用して、これらのフェイズを手動で「固定（フリーズ）」する必要があります。フェイズを固定すると、そのアクティブボーナスが15秒間ロックされ、クロックの回転が停止するため、特定のステータス上昇効果を最大限に活用できます。

### デルタクロックのフェイズと効果

| フェイズ | 色 | 基本ステータスボーナス | 最適な使用場面 |
| :--- | :--- | :--- | :--- |
| **ネイル** | 赤 | 攻撃力 +30%, ダメージ上限 +15%, クリティカル確率 +10% | 主要なDPSタイミング、リンクタイム、チェインバースト時。 |
| **オレスト** | 青 | 防御力 +40%, 怯み無効（被ダメージ時の中断耐性）, 被ダメージ -15% | ボスのオーバードライブフェイズの凌ぎ、タメ攻撃の実行時。 |
| **ゼファー** | 黄 | HP吸収 3%（1ヒットあたり最大500）, アビリティクールダウン短縮 +20% | 被ダメージ後の素早い復帰、アビリティの回転率向上時。 |

### クロックの固定と操作方法
1. **コンボフィニッシュ:** 通常攻撃コンボが終了する正確なタイミングで強攻撃ボタン（XboxではY / PlayStationでは△）を押すと、「フェイズロックストライク」を実行します。ヒットした瞬間にアクティブだったデルタクロックのフェイズが固定されます。
2. **ジャスト回避とパリィ:** ジャスト回避を成功させるか、アビリティ『スリークガード』によるカウンターを成功させると、現在のフェイズが即座に固定され、効果時間が20秒に延長されます。
3. **手動切り替え:** ベアトリクス独自のキャラクターアクションにより、奥義ゲージを10%消費してクロックを即座に次のフェイズへと進めることができます。これにより、自然な8秒の待機時間をスキップできます。

## ベアトリクスのアビリティ解説

バフの稼働率を維持し、ダメージ出力を最大化するためには、Endless Ragnarokにおけるベアトリクスのアビリティを適切に組み合わせることが極めて重要です。彼女のアビリティはデルタクロックのアクティブ状態と直接連動しており、クロックが回転中か固定中かによって、ダメージ特性、クールダウン、またはユーティリティ効果が変化します。

### 主要アクティブアビリティ

*   **ライジングブラスト (クールダウン: 18秒):** 前方への素早い多段斬撃。**ネイル**フェイズ中に実行すると、最終段のダメージが50%増加し、対象の防御力を20%無視します。ベアトリクスの主力ダメージ源となるアビリティです。
*   **サンライトブレード (クールダウン: 25秒):** 前方に突進し、炎の衝撃波を放ちます。**オレスト**フェイズ中にこのアビリティを使用すると、10秒間、自身の最大HPの20%に相当するバリアを獲得します。
*   **イモータルアサルト (クールダウン: 45秒):** 現在のデルタクロックのフェイズを消費して放つ、高威力の物理斬撃。引き換えに、ベアトリクスは6秒間の無敵状態を獲得します。このアビリティのクールダウンは、デルタクロックが固定されていた時間1秒につき2秒短縮されます。
*   **スリークガード (クールダウン: 22秒):** カウンター構えアビリティ。この構え中に攻撃を受けると、攻撃をパリィして火属性の特大ダメージを与え、即座に現在のデルタクロックのフェイズを固定します。

```
   [敵の攻撃] -> スリークガード発動 -> カウンター発生 -> デルタクロックが即座に固定
```

### アビリティデータとスケーリング

| アビリティ名 | 基本クールダウン | ダメージ倍率 | デルタクロック相乗効果 |
| :--- | :--- | :--- | :--- |
| **ライジングブラスト** | 18秒 | 4.5倍 | **ネイル**フェイズ中、ダメージ+50%および防御力20%無視。 |
| **サンライトブレード** | 25秒 | 3.8倍 | **オレスト**フェイズ中、最大HPの20%のシールドを獲得。 |
| **イモータルアサルト** | 45秒 | 8.0倍 | フェイズを消費して6秒間の無敵を獲得。固定時間に応じてクールダウンが短縮。 |
| **スリークガード** | 22秒 | 5.0倍 (カウンター) | カウンター成功時にアクティブなフェイズを即座に固定。 |

エンドゲームコンテンツ向けのアビリティ構成やクールダウン管理の最適化について詳しく知りたい方は、こちらの[武器限界突破ガイド](/ja/weapon-transcendence/)をご覧ください。

## Endless Ragnarok ベアトリクス最強ビルド：「クリムゾン・テンペスト」編成

このEndless Ragnarokベアトリクス向けビルドは、難易度「Chaos」のクエストや[ザ・コンフリュクス完全ガイド](/ja/conflux/the-conflux-complete-guide/)の深層攻略など、ハイエンドなエンドゲームコンテンツ向けに設計されています。S層のジーンを使用して通常のダメージ上限を突破しつつ、**ネイル**フェイズ中のダメージ出力を最大化します。

### 武器の選択：エムブラスクの限界突破
このビルドでは、武器限界突破で少なくともレベル10まで強化した**エムブラスク**（究極武器）を選択する必要があります。これにより、スロットII（最大レベル25の加護）とスロットIV（最大レベル10の加護）が解放され、重要な攻撃的パークを組み込むことが可能になります。

*   **スロットII加護:** クリティカル確率 (最大レベル25)
*   **スロットIV加護:** ウォーパス (最大レベル10)

### 推奨ジーン編成

ベアトリクスの限界を超えるためには、キャラクター専用ジーンと、本拡張コンテンツで追加された新しいS層ジーンを組み合わせる必要があります。

| ジーンスロット | ジーン名 | ティア | 主な効果 |
| :--- | :--- | :--- | :--- |
| **スロット1** | エムブラスクの矜持+ | S | デルタクロックの回転時間を2秒短縮。クロック固定時に攻撃速度が15%上昇。 |
| **スロット2** | デルタアクセラレーション+ | S | オレストまたはゼファーフェイズ中、アビリティのクールダウン回復速度が20%上昇。 |
| **スロット3** | 有利属性変換+ | S | すべての攻撃が敵の弱点属性になり、ダメージ上限が10%上昇。 |
| **スロット4** | クリティカル威力 V+ | A | クリティカルダメージが50%上昇（サブ効果にクリティカル確率Vを推奨）。 |
| **スロット5** | アルファ V+ | A | 奥義ダメージが上昇し、パーティ全体の奥義ゲージ上昇量を強化。 |
| **スロット6** | ガンマ V+ | A | 奥義発動時にシールドを生成。全体のダメージ上限が5%上昇。 |

### コンボ・ローテーションの手順
1. **開幕:** 手動切り替え（奥義ゲージ消費）または基本コンボを使用して、デルタクロックを**ネイル（赤）**フェイズに合わせることから戦闘を開始します。
2. **固定:** 強攻撃のフェイズロックストライクで終わる基本コンボを実行してネイルフェイズを固定し、攻撃力+30%とダメージ上限+15%のボーナスを発動させます。
3. **バースト:** 即座に**ライジングブラスト**を放ち、続けて**サンライトブレード**を使用します。ネイルフェイズが固定されているため、両アビリティとも最大のダメージ上限に達します。
4. **防御への移行:** ボスが強力な範囲攻撃（AoE）の予兆を見せたら、**スリークガード**を使用してカウンターを狙います。これにより固定時間がリフレッシュされます。スリークガードがクールダウン中の場合は、**イモータルアサルト**を使用してフェイズを消費し、6秒間の無敵を獲得して攻撃をやり過ごします。
5. **クールダウン回収:** アビリティがクールダウン中でHPが低下している場合は、クロックを**ゼファー（黄）**に切り替えて固定し、素早くHPを回復させつつアビリティのクールダウンを短縮します。

## マスタートラッカーと武器限界突破の進行

ベアトリクスの真の力を引き出すには、レベル100到達後にマスタートラッカー（Master Traits）のルートを進める必要があります。これらのトラッカーは、**インサイト**、**エッセンス**、**クラックス**の3つの異なるルートに分かれています。

```
               [ レベル100 ベアトリクス ]
             /              |              \
            /               |               \
     [ インサイト ]    [ エッセンス ]    [ クラックス ]
     (DPS/クリティカル) (便利機能/CD短縮)   (防御/奥義)
```

### ベアトリクスのマスタートラッカー・ルート
*   **インサイトルート:** クリティカル確率、クリティカルダメージ、および物理攻撃力の強化に特化しています。インサイトルートのランク3を解放すると、ベアトリクスは「エムブラスクのエッジ」を獲得し、HPが50%未満のときに物理ダメージ上限がさらに10%上昇します。
*   **エッセンスルート:** 利便性に特化しており、アビリティのクールダウン短縮や、固定されたデルタクロックの持続時間を延長します。ランク3のパーク「クロックワークマスター」は、すべての固定フェイズの持続時間を3秒延長します。
*   **クラックスルート:** 防御ステータス、奥義ゲージ上昇量、およびリンクタイムの効率に特化しています。ランク3のパーク「レジリエントソルジャー」は、オレストフェイズ中にベアトリクスに永続的な怯み無効を付与するため、ジーンスロットを1つ節約できます。

武器限界突破やマスタートラッカーに必要な高級素材を集めるには、プレイヤーは[フェイトブレイカークエスト攻略ガイド](/ja/endgame/fatebreaker-quests-guide/)を周回する必要があります。これらの高難易度コンテンツでは、エムブラスクをレベル100以上に強化するために必須となる「ラグナロク脱結晶化コア」や「フェイトブレイカーエンブレム」などのアイテムがドロップします。

## 戦闘戦略とチームシナジー

ベアトリクスは、戦場をコントロールできるチームにおいて真価を発揮します。敵の動きを止めることで、妨害されることなくその場に留まり、高火力のコンボを出し切ることができるからです。*ラグナリア*や*シエテ*といったエンドゲームのボスに挑む際は、チームのサポート役やクラウドコントロール（行動妨害）役との連携が不可欠です。

### ボス別攻略戦略

*   **ベルゼバブ:** ベルゼバブのブラックホールフェイズ中は、防御力上昇と怯み無効効果を得るために**オレスト（青）**フェイズを使用します。これにより、ベアトリクスが引き寄せられるのを防ぎ、強攻撃のタメ中に攻撃を中断されるのを防ぎます。
*   **ザ・ワールド:** ザ・ワールドは頻繁に属性を変化させ、テレポートを行います。テレポートからの強襲に対しては**スリークガード**でカウンターを狙いましょう。カウンターが成功すればアクティブなフェイズが固定され、ボスが再出現した瞬間に即座に**ライジングブラスト**で反撃できます。

### チームシナジー

ベアトリクスは、敵を拘束できるキャラクターや継続的な回復を提供できるキャラクターと非常に相性が良く、低HP維持のDPS編成を安全に運用できます。

1.  **サンダルフォン:** 現在のメタにおいて満場一致でSSティアに位置するサンダルフォン（詳細は[サンダルフォン攻略ガイド](/ja/characters/sandalphon-guide/)を参照）は、比類なきユーティリティを提供します。敵にスロウを付与してボスを拘束する彼の能力は、ベアトリクスが反撃を恐れることなく**ネイル**フェイズを固定し、フルDPSローテーションを叩き込むための完璧な隙を作り出します。
2.  **カリオストロ:** カリオストロは重要な攻撃・防御バフを提供しますが、それ以上に彼女の回復能力がベアトリクスのアグレッシブなプレイを支えます。カリオストロの「ファンタズマゴリア」とベアトリクスのデルタクロックバフの相乗効果により、パーティ全体が安定してダメージ上限に到達できるようになります。
3.  **マジェリー:** 新しいDLCサポートキャラクターであるマジェリーは、浮遊するシールドやバリアを提供します。これにより、ベアトリクスは難易度「Chaos」で一撃死するリスクを冒すことなく、マスタートラッカー「エムブラスクのエッジ」による低HP時のダメージボーナスを維持できます。

キャラクターのランキングや、ベアトリクスと他のキャラクターとの比較についてさらに詳しく知りたい方は、こちらの[Endless Ragnarok キャラクター最強ティアリスト](/ja/tier-list/)をご覧ください。

## よくある質問（FAQ）

### Endless Ragnarokでベアトリクスを解放するにはどうすればよいですか？
ベアトリクスを解放するには、まず難易度「Proud」のクエスト「プロトバハムート戦（The Tale of Bahamut's Rage）」をクリアしてエピローグを解放する必要があります。クリア後、新しいDLCクエストから入手できる「キャラクター加入チケット」をシードホルンのよろず屋シェロで交換できます。

### オレストフェイズを利用して、ベアトリクスをタンクとして運用することはできますか？
オレストフェイズは防御力+40%と怯み無効を提供しますが、ベアトリクスは根本的にはDPSキャラクターです。防御フェイズはボスの強力なギミックを生き延びたり、タメ攻撃を安全に実行したりするために使用すべきであり、メインタンクとして機能させるためのものではありません。本格的なタンク役としては、ヴェインやジークフリートといったキャラクターの方がはるかに適しています。

### ベアトリクスの武器限界突破素材を最も効率よく集める方法は何ですか？
最も効率的な方法は、**ザ・コンフリュクス**のソロモードでゲートウェイを周回することです。シードホルンやトレダーム聖域のゲートウェイを探しましょう。戦闘部屋やレアモンスター部屋のクリアに集中することで、20分間のランあたりで最も多くの強化素材を獲得できます。

### ベアトリクスのデルタクロックバフは、通常のジーンバフと重複しますか？
はい。デルタクロックによる攻撃力と防御力のボーナスは乗算され、「攻撃力 V」や「防御力 V」などの標準的なジーンと重複します。ただし、ゲーム全体のダメージ上限の制限は受けるため、ネイルフェイズの恩恵を最大限に受けるには「有利属性変換+」や「ガンマ」ジーンの装備が必要になります。
