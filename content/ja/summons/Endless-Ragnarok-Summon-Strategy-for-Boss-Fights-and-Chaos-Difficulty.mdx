2026年7月9日にリリースされた『グランブルーファンタジー リリンク』の拡張DLC「エンドレス・ラグナロク」では、極限の難易度「ケイオス」が追加され、最適化されたレベル100のビルドでさえも苦戦を強いられています。これらの高難易度エンカウンターを生き抜くためには、新しく実装された「召喚システム」のマスターが不可欠です。エンドゲームクエストで頻発する一撃必殺（即死）ギミックを緩和するには、最適化された**エンドレス・ラグナロクの難易度「ケイオス」向け召喚獣**セットアップを活用することが鍵となります。星晶獣を呼び出すルリアの能力は、パッシブなステータス上昇だけでなく、戦況を覆すことができる操作可能なアクティブエンティティをもたらします。

適切な**エンドレス・ラグナロクのボス戦向け召喚獣**を選ぶには、ボスのギミック、チーム編成、そしてプレイヤーの役割を分析する必要があります。難易度「ケイオス」において、ベルゼバブ、「ザ・ワールド」、ラグナリアといった敵は膨大なHPと激しいフェーズ移行を持ち、正確な召喚タイミングが要求されます。本ガイドでは、召喚グリッドの構築方法、マルチプレイ（コープ）でのアクティブ召喚の連携方法、および重要なリンクタイム中に大ダメージを与える「プライマルバースト」の実行方法について詳しく解説します。

## 難易度「ケイオス」における召喚システムの仕組み

召喚システムは進行状況によってロックされています。プレイヤーはまず、PROUD難易度のクエスト「The Tale of Bahamut's Rage」をクリアしてエピローグを解放し、難易度「ケイオス」のコンテンツにアクセスする必要があります。解放されると、プレイヤーはロードアウトに最大4体の召喚獣を装備できるようになります。ただし、召喚獣をアクティブに具現化（召喚）できるのは、フェイトブレイカー（Fatebreaker）クエストや、[ローグライトモード「ザ・コンフラックス」](/ja/conflux/the-conflux-complete-guide/)の深層など、難易度「ケイオス」以上に指定されたクエスト中に限られます。

バトルには4体の召喚獣を持ち込めますが、4人パーティー全体で同時にフィールドに出せるアクティブ召喚獣は常に1体のみです。召喚を発動するとパーティー全体で共有されるクールダウンタイマーが開始され、プレイヤーが同時に複数のアクティブ召喚を連発するのを防ぎます。召喚獣がフィールド上でアクティブになっていない間も、装備しているキャラクターにパッシブなステータス上昇効果を提供するため、ダメージ上限や生存ラインに到達するためには**エンドレス・ラグナロクの召喚獣最適化**が極めて重要になります。

| 召喚システム・メカニクス | システム詳細と数値 | 戦闘での応用 |
| :--- | :--- | :--- |
| **召喚グリッド枠** | キャラクターごとに4体 | パッシブボーナスを提供（例：攻撃力、クリティカルダメージ、HP） |
| **アクティブ具現化** | グローバルで同時に1体のみ | 同時に召喚獣を操作できるのはプレイヤー1人のみ |
| **アクティブ持続時間** | 45秒 | 召喚獣がフィールドに留まり、アクションを実行するかプレイヤーに操作される |
| **パーティー共有クールダウン** | 120秒 | 他のパーティーメンバーによる連続した即時召喚を防ぐ |
| **プライマルバーストのトリガー** | リンクタイム中の4人連携SBA | 装備している召喚獣に基づいた強力な無属性ダメージを与える |

装備した召喚獣のパッシブボーナスは、フェイトブレイカークエストや武器限界突破（Weapon Transcendence）チャレンジから入手できる素材を使用して強化レベルを上げることでスケールします。例えば、バハムートやティアマト・マグナなどの上位召喚獣を装備するとキャラクターのダメージ上限が上昇し、ナルメアやゼタのようなSランクキャラクターがDPSの限界を突破できるようになります。

## 操作可能な召喚獣：ルシファーとローラン

召喚可能な星晶獣のラインナップの中で、ルシファーとローランは、具現化時にプレイヤーが直接操作できる唯一の2体の召喚獣として際立っています。プレイヤーがルシファーまたはローランを召喚すると、バトルフィールド上のキャラクターが一時的に召喚獣と入れ替わり、プレイヤーが直接その召喚獣を操縦できるようになります。これは、効果的な**エンドレス・ラグナロクの召喚獣エンドゲーム編成**における中核要素です。

### ルシファー（攻撃特化型）
ルシファーは純粋なアタッカーとして機能します。彼の通常攻撃は広範囲に薙ぎ払う光の波動を放ち、複数のヒットボックスを貫通するため、「ザ・ワールド」のような大型ボスに対して非常に効果的です。彼の代表的なスキル「パラダイス・ロスト」は、広範囲に強力な光属性ダメージを与え、対象に一時的な防御力ダウンのデバフを付与します。天司長モードのサンダルフォンのようなトップクラスのDPSキャラクターを使用している場合、ボスのブレイク（Break）中にルシファーを召喚することで、与えるダメージを最大化できます。このセットアップを補完するためのサンダルフォンのビルド最適化については、[サンダルフォンビルドガイド](/ja/builds/sandalphon-build/)で詳しく解説しています。

### ローラン（防御・ユーティリティ支援型）
ローランはパーティーの生存力とクラウドコントロール（敵の妨害）に特化しています。操作時、ローランはパーティーメンバーの最大HPの50%に相当する被ダメージを吸収する巨大な保護バリアを展開できます。彼の主な攻撃は高いスタン値を与え、ラグナリアのようなボスのオーバードライブシールドを破壊するのに不可欠です。ローランのアルティメットスキルは、アクティブなパーティーを回復し、一時的な弱体効果（デバフ）無効化を付与します。これは、ボスが「即死」や「氷結」といった致命的なデバフを付与してくる難易度「ケイオス」の戦闘において極めて価値があります。

| 属性 / スキル | ルシファー | ローラン |
| :--- | :--- | :--- |
| **主要属性** | 光 | 風 / 光 |
| **通常攻撃スタイル** | 遠距離光波斬撃（高DPS） | 近接剣撃（高スタン） |
| **特殊能力 I** | パラダイス・ロスト（広範囲大ダメージ＋防御ダウン） | セイクリッド・シールド（パーティーにHP50%分のバリア） |
| **特殊能力 II** | ポースポロス（単体高怯み攻撃） | ファウンテン・オブ・ライト（全体回復＋デバフ無効） |
| **最適な使用タイミング** | ボスのブレイク時 / ダメージチャンス | ボスのオーバードライブ時 / フェーズ移行期 |

## エンドゲームボス向けの上位召喚獣ロードアウト

最適な**エンドレス・ラグナロクの召喚獣組み合わせ**を構築するには、エンドゲームボスがもたらす特定の脅威に召喚グリッドを合わせる必要があります。難易度「ケイオス」のボスは、準備を怠ったパーティーを数秒で全滅させる独自のギミックを持っています。

例えば、ベルゼバブは自身の攻撃速度とダメージを上昇させるバフを頻繁に付与します。これに対抗するため、少なくとも1人のパーティーメンバーがディスペル（強化効果消去）効果を持つ召喚獣を装備するか、防御デバフを付与できるカリオストロのようなキャラクターを起用する必要があります。また、「ザ・ワールド」は、現実を歪める全滅ギミックを阻止するために高い怯み（スタッガー）値を要求するため、スタン特化の召喚獣が非常に有効です。

```
[装備中の召喚獣] ──> パッシブステータス上昇（常時発動）
       │
       └───> アクティブ召喚（共有クールダウン） ──> 戦術的展開（ブレイク/オーバードライブ）
```

以下の表は、難易度「ケイオス」における3大エンドゲームボスに対する最適な召喚獣ロードアウトを示しています。

| 対象ボス | 推奨アクティブ召喚獣 | 推奨パッシブグリッド召喚獣 | 戦術的実行戦略 |
| :--- | :--- | :--- | :--- |
| **ベルゼバブ** | ルシファー | バハムート、プロトバハムート、ティアマト、コロッサス | ベルゼバブの「ケイオス・カラミティ」フェーズの直後にルシファーを召喚し、シールドを削りつつ防御ダウンを付与する。 |
| **ザ・ワールド** | ローラン | ユグドラシル、リヴァイアサン、マキュラ・マリウス、アニマ | ザ・ワールドがエリア全域にレーザーを照射し始めたら、ローランのセイクリッド・シールドを展開して低HPのDPSキャラクターを保護する。 |
| **ラグナリア** | ローラン / ルシファー | ゼピュロス、アグニ、シヴァ、グリームニル | ラグナリアのシールドフェーズ中にローランを回してスタンダメージを与え、その後ルシファーで追撃してブレイク中のダメージを最大化する。 |

これら上位召喚獣の強化素材を入手するには、高難易度コンテンツを周回する必要があります。武器と召喚獣を同時に強化するための効率的な周回ルートの詳細は、[武器限界突破（Weapon Transcendence）強化ガイド](/ja/weapon-transcendence/weapon-transcendence-guide/)を参照してください。

## リンクタイム中のプライマルバーストの発動

プライマルバーストは、召喚システムに関連する究極の攻撃ギミックです。プライマルバーストを発動するには、まずリンクレベルを100%にしてリンクアタックを行い、リンクタイムに突入する必要があります。リンクタイムが発動したら、4人のパーティーメンバー全員が奥義（SBA）を連続してチェインさせる必要があります。

4人全員がアクティブスロットに召喚獣を装備している場合、チェインバーストが「プライマルバースト」へとアップグレードされます。プライマルバーストのダメージは、装備している召喚獣の属性の組み合わせによってスケールします。召喚獣の属性をボスの弱点属性に合わせることで、プライマルバーストのダメージが最大40%上昇します。

### プライマルバースト発動のステップ・バイ・ステップ：
1. **リンクレベルを溜める**: 連携攻撃を行い、ボスのギミックを回避してリンクレベルを100%にします。
2. **リンクタイムを発動する**: 強攻撃やスタンフェーズの後に表示されるプロンプトに従ってリンクアタックを実行します。
3. **奥義（SBA）を繋ぐ**: 最初のプレイヤーがSBAを発動します。残りの3人のプレイヤーは、前の演出が終わる前にSBAをチェインさせる必要があります。
4. **プライマルバースト発動**: 4回目のSBAが完了すると、装備された召喚獣たちが具現化して同期された属性攻撃を実行し、ボスの防御値を完全に無視してダメージを与えます。

| 召喚獣の属性組み合わせ | プライマルバーストの種類 | ダメージ倍率 | 追加ステータス効果 |
| :--- | :--- | :--- | :--- |
| **同一属性×4** | モノエレメンタル・カタクリズム | 2.5倍 | 属性耐性ダウン（-20%）を付与 |
| **光×2 ＋ 闇×2** | トワイライト・ルイン | 2.2倍 | ボスに8秒間のスロウを付与 |
| **異なる4属性** | プライマル・ケイオス | 2.0倍 | ランダムで火傷、毒、または麻痺を付与 |

## マルチプレイにおける連携とローテーション戦略

アクティブ召喚はパーティー全体で120秒のグローバルクールダウンを共有するため、マルチプレイでのコミュニケーションが極めて重要です。連携せずにランダムに召喚を行うと、クールダウンが無駄になり、重要なボスフェーズで何もできない時間が生じてしまいます。体系的な召喚ローテーションを組むことで、ボスのアルティメット攻撃中に防御バリアを展開し、ダメージチャンス時に攻撃的な召喚獣を配置することができます。

標準的なローテーション戦略では、各プレイヤーに役割を割り当てます。
* **プレイヤー1（メインDPS、例：サンダルフォンやナルメア）**: ルシファーを装備。最初のブレイクフェーズ中の召喚を担当。
* **プレイヤー2（サポート/タンク、例：マギラフリラやヴェイン）**: ローランを装備。ボスがオーバードライブに入った際に召喚し、大ダメージを吸収する役割を担当。
* **プレイヤー3（スタン特化、例：バザラガやシャルロッテ）**: コロッサスなどの高スタンパッシブ召喚獣を装備。
* **プレイヤー4（ユーティリティ、例：カリオストロやユーステス）**: デバフ解除やバフ付与を行うユーティリティ召喚獣を装備。

```
[ボスフェーズ：オーバードライブ] ──> プレイヤー2がローランを召喚（セイクリッド・シールド）
       │
[ボスフェーズ：ブレイク]         ──> プレイヤー1がルシファーを召喚（パラダイス・ロスト）
       │
[リンクタイム発動]               ──> 全員がSBAを開始 ──> プライマルバースト
```

以下のタイムラインは、典型的な6分間の難易度「ケイオス」ベルゼバブ戦における最適な召喚ローテーションを示しています。

| 時間 | ボスフェーズ | 必要なアクション | 展開するアクティブ召喚獣 |
| :--- | :--- | :--- | :--- |
| **0:45** | 通常フェーズ | リンクレベルとSBAゲージを溜める | なし（クールダウンを温存） |
| **1:30** | オーバードライブフェーズ1 | ボスが「ブラック・フライズ」（強力な範囲攻撃）を詠唱 | **ローラン**（プレイヤー2がバリアを展開してダメージをブロック） |
| **3:00** | ブレイクフェーズ1 | ボスが行動不能になり、防御力が低下 | **ルシファー**（プレイヤー1が召喚し、最大DPSを叩き出す） |
| **4:30** | オーバードライブフェーズ2 | ボスが「ケイオス・カラミティ」を詠唱 | **ローラン**（プレイヤー2が召喚し、回復とデバフ解除を行う） |
| **5:15** | リンクタイム / SBA | フィニッシュに向けて4人チェインを実行 | **プライマルバースト**（すべての召喚獣が具現化） |

ソロプレイでこれらのローテーションを試したいプレイヤーは、AIの仲間がリンクタイム発動までSBAを温存するように設定できます。これにより、連携の取れたマルチプレイパーティーがなくても、確実にプライマルバーストを発動できます。AIパーティーメンバーの設定やマスタートレイト（Master Traits）の解放に関する詳細は、[マスタートレイト進行ガイド](/ja/master-traits/master-traits-explained/)を参照してください。

パッチノート、召喚獣のバランス調整、コミュニティイベントに関する最新情報は、公式の[Cygames Granblue Fantasy: Relink ウェブサイト](https://relink.granbluefantasy.jp/en/)をご確認ください。

## よくある質問

### PROUD難易度のクエストでアクティブ召喚を使用できますか？
いいえ。アクティブ召喚は、難易度「ケイオス」以上に指定されたクエストでのみ具現化できます。ただし、装備した召喚獣からのパッシブステータスボーナスは、PROUDやMANIACを含むすべての難易度帯で有効です。

### 操作可能な召喚獣であるルシファーとローランはどのように解放しますか？
ルシファーとローランは、拡張DLC「エンドレス・ラグナロク」の本編エピローグをクリアした後に、それぞれのフェイトブレイカークエストをクリアすることで解放されます。これらのクエストにアクセスするには、「The Tale of Bahamut's Rage」をクリアしている必要があります。

### 同じ種類の召喚獣を複数装備した場合、パッシブボーナスは重複しますか？
はい、パッシブボーナスは重複しますが、キャラクターのステータス上限を超えると減衰効果が適用されます。そのため、同じ召喚獣を4体重ねるよりも、多様なグリッド（例：HP上昇1、攻撃力上昇1、クリティカルダメージ上昇2など）を構築することをお勧めします。

### アクティブ召喚獣が倒された場合のクールダウンペナルティはありますか？
ルシファーやローランなどのプレイヤー操作の召喚獣が、45秒の持続時間が経過する前に致命的なダメージを受けた場合、その召喚獣は消滅し、ペナルティとしてパーティー共有のクールダウンが120秒から180秒に延長されます。召喚獣を操作している間も、安全な立ち回りと回避が必要です。
